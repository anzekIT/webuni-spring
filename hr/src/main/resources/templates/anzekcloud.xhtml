Szolgatatoi adatok

  • SzerverIdo = ServerTime
  • RendszerTerulet = SystemSize
  • Szolgaltato : Anzek Informatika Kft
  • Savszelesseg : 1GB / sec
  • A szolgaltatas hozzaferese : kizarolag az "LSZMG ERP" rendszer elofizetoi szamara!
  • Technikai fuggosege : az "LSZMG ERP" rendszer es egyes kliensei automatikus eleressel
  • Bongeszobol inditott manualis lekresek inditasa = nem lehetseges, nem megoldhato!

LSZMG CLOUD API EndPoints (az ERP szolgaltatasok VegPontjai)

(Figyelem! Az alabbi vegpontokra titkositott, szoftveres kereseket var a backEndKiszolgalo!)
  • .../api/anzekcloud/techinfo = a felho tarhely (a backend kiszolgalo es az adatbazis) adatai
  • .../api/anzekcloud/userinfo = a felhasznalo nyilvatartott adatai
  • .../api/anzekcloud/softwareinfo = a backend kiszolgalo- es az adatbazisverziok adatai
  • .../api/anzekcloud/consumer/product = termelesi adatok
  • .../api/anzekcloud/consumer/resorces = gyartasi- es egyeb eroforras adatok
  • .../api/anzekcloud/consumer/warehouse = raktari adatok
  • .../api/anzekcloud/consumer/stocks = keszletezesi adatok
  • .../api/anzekcloud/consumer/sales = ertekesitesi adatok
  • .../api/anzekcloud/consumer/invoices = kimeno-szamlak adatai
  • .../api/anzekcloud/consumer/incoming = beerkezo keszletek adatai
